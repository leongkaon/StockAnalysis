Date: 20210405  
Description:  
早排同筆友傾計，佢提到「股市得1/4時間係升」，我地可以證實下係唔係，順手熟習Python。  
假設以2000年至近日(2020年4月1)，以每日收盤價為基準計算，以「今日」減「上個收市日」是否大於0 判斷係升或跌  
號碼我揀左0388.hk，應該可以代表整體香港股市?  

結論: 無考慮升跌百份比情況下，單純以「升」或「跌」判斷，港交所(0388.hk)有2414日係升，2434日係跌。升:跌大約0.99:1。
